# poe2guide-content

Публичные данные для Android-приложения **PoE 2 Гайды** (справочник по билдам Path of Exile 2).

Приложение читает `builds.json` по «сырой» ссылке и кэширует его, чтобы обновлять
билды и мету **без пересборки и переустановки** приложения.

## Файлы
- `builds.json` — список билдов (название, класс, восхождение, тир, патч, краткая сводка, ключевые скиллы, ссылка на полный гайд).

## Как обновить контент
Достаточно отредактировать `builds.json` и закоммитить — приложение подтянет свежую версию при следующем запуске (при наличии сети).

## Источник полных гайдов
Полные гайды — на [maxroll.gg](https://maxroll.gg/poe2/build-guides). Здесь хранятся только краткие структурированные выжимки и ссылки, без копирования авторских статей.
